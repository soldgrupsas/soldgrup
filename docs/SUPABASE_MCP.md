## Acceso al servidor MCP de Supabase desde este proyecto

El servidor MCP de Supabase ya está activo en esta instancia de Cursor, por lo que no necesitas credenciales ni configuración adicional en este repositorio para usarlo.

### ¿Qué puedes hacer?
- **Listar organizaciones**
- **Listar proyectos**
- **Consultar el estado de un proyecto**
- **Ver logs** (API, Postgres, Auth, Storage, Realtime)
- **Ejecutar SQL** o **aplicar migraciones**
- **Gestionar funciones Edge**

### Uso rápido (desde la IA en Cursor)
Pide a la IA acciones concretas, por ejemplo:

- "Lista mis organizaciones de Supabase"
- "Muestra los proyectos del org con ID <org_id>"
- "Ejecuta este SQL en el proyecto <project_id>"
- "Obtén los logs de Postgres del proyecto <project_id>"

La IA usará el servidor MCP de Supabase conectado para ejecutar estas operaciones de manera segura.

### Identificadores útiles detectados
- Organización: `Soldgrup` (ID: `pekrifvxhyvocbupmhqu`)

### Notas
- Si en algún momento necesitas portar esta configuración a otra máquina donde el servidor MCP no esté preconfigurado, se puede añadir un archivo `.cursor/mcp.json` con la definición del servidor MCP de Supabase. En esta instancia no es necesario porque el servidor ya está activo globalmente en Cursor.
